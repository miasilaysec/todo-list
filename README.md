# To-Do List

HTML, CSS ve JavaScript ile gelistirilmis basit bir to-do list uygulamasi.

## Ozellikler

- Gorev ekleme
- Gorev duzenleme
- Gorev silme
- Arama ile filtreleme
- Durum yonetimi (Yapilacak, Devam Ediyor, Tamamlandi)
- Verilerin localStorage ile saklanmasi

## Calistirma

`index.html` dosyasini tarayicida acman yeterlidir.
